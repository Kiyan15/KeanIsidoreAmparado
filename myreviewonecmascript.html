Review about the history of EcmaScript

The ECMAScript standard is the foundation for languages like Javascript, and ECMAScript is a subset of JavaScript. At its foundation, JavaScript is ECMAScript, although it extends it. ECMAScript is a scripting language standard. As an example, AS/JS/JScript are three distinct automobiles, yet they all share the same engine. Their exteriors, however, are distinct, and each has undergone various modifications to make it unique.

ECMAScript = ES:

JavaScript is a general-reason scripting language that conforms to the ECMAScript specification. The ECMAScript specification is a blueprint for developing a scripting language. JavaScript is an implementation of that blueprint. On the whole, JavaScript implements the ECMAScript specification as defined in ECMA-262.
